Plainscript

// medication reference

Type in a medication to find out what it actually is, or check whether two or more medications are known to interact — explained in plain language, sourced from FDA and NIH data.

Enter a brand or generic name (for example, Lipitor or atorvastatin).

Add two or more medications. Plainscript checks them against a curated set of well‑documented interactions and cross‑references each drug's official FDA label.

How this works & what it can't do

Where the data comes from ›

Drug facts come from the U.S. FDA's openFDA drug label API — the actual approved labeling for each medication (what it treats, how it works, side effects, brand/generic names, drug class).

Name matching uses the NIH/NLM RxNorm API, which standardizes drug names and tolerates typos.

Interaction severity comes from a hand‑curated set of well‑documented interactions, each reviewed against FDA prescribing information and standard clinical references.

Why there's no simple "you're safe" result ›

In January 2024 the U.S. National Library of Medicine shut down its free drug‑interaction API. There is no longer a free, comprehensive, severity‑rated interaction database — paid clinical databases (DrugBank, Micromedex) hold that data now.

So this tool can confirm an interaction it knows about, but it can never prove that a combination is safe. A result of "nothing found" only means it wasn't in our curated set or the FDA labels we checked — not that the drugs are safe together. That's why you'll never see a green check here. Always confirm with a pharmacist.

Honest limitations ›

The curated set covers a few dozen of the most important interactions, not all of them. The FDA‑label layer only catches interactions a label names directly, and labels vary in detail. Doses, your health conditions, food, and supplements all matter and aren't considered here. This is a learning tool, not a clinical decision system.
